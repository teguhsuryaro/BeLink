# Fase 01: Persiapan Repositori & Dependensi

Langkah ini adalah tahap awal untuk mengunduh seluruh kode sumber (source code) proyek BeLink ke laptopmu dan memasang semua library yang dibutuhkan.

## 1. Unduh Kode (Clone Repository)
Buka terminal (Command Prompt / PowerShell / Terminal Mac) di folder tempat kamu ingin menyimpan proyek, lalu jalankan:

```bash
git clone <URL_GITHUB_PROYEK_INI>
cd <NAMA_FOLDER_CLONE>
```

## 2. Install Dependensi (Library)
Proyek ini menggunakan Node.js dan NPM. Pastikan kamu sudah menginstal Node.js di laptopmu. Jalankan perintah berikut untuk mengunduh semua library yang dibutuhkan:

```bash
npm install
```

Tunggu hingga proses selesai. Jika sudah tidak ada error, lanjutkan ke panduan **02-persiapan-database.md**.
